# Mohammed Osman Ali — Portfolio

## Files
- index.html   -> page structure (Home, About, Skills, Projects, Contact)
- style.css    -> all colors, fonts, layout, animations (see the palette at the top)
- script.js    -> navbar scroll effect, mobile menu, typing effect, scroll animations
- profile-photo.jpg -> your photo used in the hero section

## How to edit
- Change colors: edit the ":root" section at the top of style.css
- Change the typed words in the hero: edit the "roles" array at the top of script.js
- Add a new project card: copy one ".project-card" block in index.html and edit the text + link
- Add a new skill: copy one ".skill-card" block in the Skills section

## How to view
Just double-click index.html to open it in your browser. No installation needed.

## How to publish for free (GitHub Pages)
1. Push these files to a GitHub repo
2. Go to Settings -> Pages -> set branch to "main" and folder to "/ (root)"
3. Your site will be live at https://<your-username>.github.io/<repo-name>
